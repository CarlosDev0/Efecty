La solución implementada consistió en:
Creación de una solución con Clean Architecture con las siguientes capas:
(*) Infrastructure
(*) Core
(*) Api
(*) AplicaciónWeb

Algunas características de la solución:
-Para la conexión a la base de datos se utilizó el micro ORM Dapper
-Se utilizaron procedimiento almacenados
-Se utilizó el patrón Dto
-Se utilizó inyección de dependencias
-La aplicación Web utiliza MVC y .Net Core
-La API es REST y se utilizó .Net Core
-Se implementaron las validaciones solicitadas en Javascript y JQuery
-Se realizaron pruebas con Postman
-Se implementaron principios SOLID
